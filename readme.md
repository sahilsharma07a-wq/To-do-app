# 📝 Todo App (Full Stack)

A full-stack Todo application built using **Node.js, Express, MongoDB** for backend and **Vanilla JavaScript (Axios)** for frontend.

---

##  Features

*  User Authentication (Signup & Login with JWT)
*  Create, Read, Update, Delete (CRUD) Todos
*  User-specific tasks (each user sees only their own tasks)
*  Dashboard with task summary (Total, Completed, Pending)
*  Persistent login using localStorage
*  Account section displaying user details
* Protected routes using authentication middleware

---

##  Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication

### Frontend

* HTML
* CSS
* JavaScript (Vanilla)
* Axios
